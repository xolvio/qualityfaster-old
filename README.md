# Quality Faster

### Pre-requisites
Install Meteor:
```bash
curl https://install.meteor.com/ | sh
```

Install Node v4.3 
*You might want to use [NVM](https://github.com/creationix/nvm#install-script) to do this*
```bash
nvm install v4.3
nvm use v4.3
npm install
```

### Run
Start Wallaby (use the ``.wallaby.js` configuration)

open two terminals:

Terminal 1:
```bash
npm run meteor
```

Terminal 2:
```bash
npm run meteor
```

Do BDD with your team!
